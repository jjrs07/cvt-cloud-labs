# 📊 NoSQL Databases (DynamoDB)

Learn how to handle massive scale and flexible data structures using Amazon DynamoDB, the serverless NoSQL database.

## 🗺️ Lab Roadmap

| Lab | Goal | Description |
| :--- | :--- | :--- |
| [**Lab 1: DynamoDB & Python**](./Lab%201%20-%20DynamoDB%20with%20Python/README.md) | **Integration** | Use Boto3 to store and retrieve data from a NoSQL table. |

---

## 🛠️ What You'll Learn
- Key-Value pairs and Document storage concepts.
- Defining **Partition Keys** and **Sort Keys**.
- Why DynamoDB is "Serverless" compared to RDS.
- Using Python to automate data entry.
