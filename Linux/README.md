# 🐧 Linux Fundamentals Lab

Welcome to the Linux module! This collection of labs will take you from basic CLI navigation to advanced user management and scripting.

## 🗺️ Lab Roadmap

| Lab | Title | Key Commands |
| :--- | :--- | :--- |
| [**Lab 1**](./Lab%201/README.md) | Basic Commands | `whoami`, `hostname`, `uptime` |
| [**Lab 2**](./Lab%202/README.md) | Navigation | `pwd`, `cd`, `ls` |
| [**Lab 3**](./Lab%203/README.md) | File Management | `mkdir`, `touch`, `cp`, `mv` |
| [**Lab 4**](./Lab%204/README.md) | File Content | `cat`, `head`, `tail`, `grep` |
| [**Lab 5**](./Lab%205/README.md) | Permissions (Part 1) | `chmod`, `chown` |
| [**Lab 6**](./Lab%206/README.md) | Permissions (Part 2) | Advanced permissions, `sudo` |
| [**Lab 7**](./Lab%207/README.md) | User Management | `useradd`, `groupadd`, `usermod` |
| [**Lab 8**](./Lab%208/README.md) | Disk Management | `df`, `du`, `mount` |
| [**Lab 9**](./Lab%209/README.md) | Services & Monitoring | `systemctl`, `top`, `ps` |
| [**Lab 10**](./Lab%2010/README.md) | Networking Tools | `ping`, `netstat`, `ifconfig` |
| [**Lab 11**](./Lab%2011/README.md) | Cron Jobs | `crontab -e`, `at` |
| [**Lab 12**](./Lab%2012/README.md) | Log Analysis | `/var/log`, `journalctl` |
| [**Lab 13**](./Lab%2013/README.md) | Archiving & Compression | `tar`, `zip`, `unzip` |
