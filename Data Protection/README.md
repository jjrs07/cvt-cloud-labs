# 💾 Data Protection & Backup

In the cloud, "Everything fails all the time." Learn how to automate your backups to ensure you never lose critical data.

## 🗺️ Lab Roadmap

| Lab | Goal | Description |
| :--- | :--- | :--- |
| [**Lab 1: AWS Backup**](./Lab%201%20-%20AWS%20Backup/README.md) | **Automation** | Create a centralized backup plan for EBS and RDS. |

---

## 🛠️ What You'll Learn
- Centralized management using **Backup Vaults**.
- Defining **Retention Policies** (how long to keep data).
- The difference between a Snapshot and a Backup Plan.
- Restoring a resource from a backup.
