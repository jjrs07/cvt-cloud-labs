# 🌐 Content Delivery & DNS

Speed up your applications and manage your "address on the internet" using CloudFront and Route 53.

## 🗺️ Lab Roadmap

| Lab | Goal | Description |
| :--- | :--- | :--- |
| [**Lab 1: CloudFront CDN**](./Lab%201%20-%20CloudFront/README.md) | **Performance** | Speed up an S3 static website using a global Content Delivery Network. |
| [**Lab 2: Route 53 (DNS)**](./Lab%202%20-%20Route%2053/README.md) | **Routing** | Understand DNS records (A, CNAME) and how to manage a domain. |

---

## 🛠️ What You'll Learn
- How **Edge Locations** reduce latency for global users.
- Securing your website with **HTTPS** via CloudFront.
- The lifecycle of a DNS query.
- Mapping "human-friendly" names (google.com) to IP addresses.

---

## ⚠️ Important Note
- **CloudFront:** Has a generous Free Tier (1TB of data transfer per month).
- **Route 53:** Hosted zones cost **$0.50 per month**. Domain registration costs vary. These labs are designed to be completed *conceptually* or with minimal cost.
